# Метод для приглашения пользователя в систему workonflow
```POST api.workonflow.com/1/team/invite-user/{query}```

### Query:
```js
query: {
  name: 'userName',
  email: 'userEmail'
}
```
### RESPONSES:
| code        | message | description|
|:------------- |:---------------|:----------------------|
| 200          | OK        |  |
